# Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente

Este repositório apresenta um site sobre agroecologia urbana e Agricultura Urbana, desenvolvido como parte do tema Agrinho.

## Instruções de uso

1. Abra o arquivo `index.html` em um navegador moderno.
2. Navegue pelas seções para conhecer as práticas de agroecologia urbana e exemplos de plantio na cidade.
3. Utilize o botão de modo escuro para alternar entre visual claro e escuro.
4. Preencha o formulário de contato para enviar uma mensagem fictícia ao projeto.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

## Objetivo do tema Agrinho

O objetivo deste tema Agrinho é demonstrar como a agroecologia urbana pode unir produção de alimentos e preservação ambiental,
criando oportunidades de trabalho e renda para pessoas na cidade, além de fortalecer a sustentabilidade local.
